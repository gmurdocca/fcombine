fcombine
========

Fcombine is a centrally managed, open source, database driven secure file transfer and automounter application including a web management front end. The projectâ€™s main purpose is collecting and securely re-exposing various network file systems including CIFS, SSH-SFTP and FTP/FTP-SSL (all variants). The application is written predominantly in Python with some C, and currently utilises the Django web application framework. It features support of directory service integration/authentication (LDAP/AD/RADIUS), automated web-defined file transfer jobs and hooks for in-flow 3rd party DLP/AV integration.
